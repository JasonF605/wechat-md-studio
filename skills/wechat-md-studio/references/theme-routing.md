# Theme Routing

Use this reference only when changing or explaining smart theme selection.

## Categories

- `ai-money`: AI赚钱, side hustles, monetization, automation, workflow, tools, models, prompts.
- `tutorial`: code, API, install, deploy, config, command-line walkthroughs.
- `science`: scientific explanation, research, mechanism, data, knowledge articles.
- `food`: recipes, restaurants, home cooking, food lifestyle.
- `health`: health literacy, safety, elder care, symptoms, emergency reminders.
- `essay`: personal essays, field notes, diaries, reflection, narrative posts.
- `news`: latest updates, announcements, policy, official statements, timely commentary.

## Default Themes

- `ai-money` -> `tech-pulse`
- `tutorial` -> `github-doc`
- `science` -> `science-clean`
- `food` -> `food-magazine`
- `health` -> `health-trust`
- `essay` -> `essay-paper`
- `news` -> `news-brief`
- weak signal -> `minimal-ink`

## Tuning Rule

Prefer precise content-signal keywords over broad mood words. For example, `自动化`, `工作流`, and `变现` are better `ai-money` signals than `高级感`; `症状`, `医生`, and `急救` are better `health` signals than `温暖`.
